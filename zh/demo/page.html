                  # 页面结构

                  此页面应当包含:

                  • 路径导航open in new window
                  • 标题和页面信息open in new window
                  • TOC (文章标题列表)open in new window
                  • 贡献者、更新时间等页面元信息open in new window
                  • 评论open in new window
                  • 导航栏open in new window
                  • 侧边栏open in new window
                  • 页脚open in new window
                  • 返回顶部按钮

                  你可以通过主题选项和页面 Frontmatter 自定义它们。
